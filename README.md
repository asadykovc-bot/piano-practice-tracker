# Piano Practice Tracker

A small PWA for tracking piano practice time with a stopwatch, metronome, local history, weekly/monthly stats, and JSON import/export.

## Files

- `index.html` - app markup
- `styles.css` - visual design
- `app.js` - timer, metronome, history, import/export, wake lock
- `manifest.json` - PWA metadata
- `service-worker.js` - offline cache
- `icon.svg`, `icon-192.png`, `icon-512.png` - app icons

## GitHub Pages

Upload all files to the root of the repository, then enable GitHub Pages:

1. Open repository `Settings`.
2. Open `Pages`.
3. Choose `Deploy from a branch`.
4. Select branch `main` and folder `/root`.
5. Save.

The app will be available at:

`https://asadykovc-bot.github.io/piano-practice-tracker/`
